Overview

Purpose of the Guide

The User's Guide helps non-technical users understand how to operate the system safely and correctly. It focuses on visible pages, common workflows, role permissions, field validation, and the expected result after each action.

The system centralizes health insurance company records, policy details, employee information, employee policy requests, manager approval, finance payment, and administrative reporting.

Access Control

User Roles

Admin

Controls companies, policies, employees, direct assignments, requests, search, and reports.

Employee

Manages personal details, searches policies, orders insurance, and changes password.

Manager

Reviews requests and bills, approves or rejects them, and forwards approved bills to finance.

Finance Manager

Credits forwarded bill payments and closes completed requests after payment.

Public Visitor

Can view Home, About, Contact, Privacy, Terms, and Sitemap pages before logging in.

Security Rule

Users do not select a role during login. The system finds the role from saved account data.

First Steps

Getting Started

Open the Application

  1. Open the system URL in a modern browser.
  2. Use the public navigation to read Home, About, and Contact information.
  3. Click the Login button in the top navigation.

General Navigation

  1. After login, use the role sidebar to move between dashboard pages.
  2. Use buttons with icons for add, edit, delete, approve, reject, pay, or close actions.
  3. Use Logout from the sidebar when work is complete.

Account Access

Login and Password Recovery

Login Steps

  1. Enter the username.
  2. Enter the password.
  3. Submit the login form.
  4. The system redirects to the correct dashboard based on the saved user role.

Forgot Password Steps

  1. Click forgot password from the login page.
  2. Enter the registered email address.
  3. Open the reset link sent to email.
  4. Enter and confirm a new strong password.

Password Rule

Password entries should use at least 8 characters with a letter, number, and symbol.

Role Manual

Admin User Instructions

Companies and Policies

  1. Open Companies to view insurance company records.
  2. Use Add Company to enter name, address, contact, email, website, and description.
  3. Open Policies to add policy name, company, policy type, premium, coverage, and duration.
  4. Use edit, activate, deactivate, or delete actions as needed.

Employees and Assignments

  1. Open Employees to review registered staff.
  2. Use Register Employee to create login details and assign a role.
  3. Use Assign Policy to directly link an active policy to an employee.
  4. Use Policy Requests to approve or reject employee insurance requests.

Role Manual

Employee User Instructions

Step 1

Dashboard

View profile summary, assigned policies, and recent requests.

Step 2

Details

Review personal details and current employee information.

Step 3

Search

Filter policies by type, coverage, and premium amount.

Step 4

Order

Submit request reason and bill amount for a selected policy.

Step 5

Status

Return to dashboard to monitor pending, approved, paid, closed, or rejected status.

Role Manual

Manager User Instructions

  1. Open Dashboard or Requests to see received bills and policy requests.
  2. Open Request Details to inspect employee, policy, bill amount, and request reason.
  3. Choose approve or reject and add remarks where needed.
  4. Forward approved bills to the Finance Manager for payment processing.
  5. Use My Details and Update Details to maintain manager profile information.

Role Manual

Finance Manager Instructions

  1. Open Dashboard or Requests to view bills forwarded by managers.
  2. Review request details and bill amount before payment.
  3. Use Credit Payment only when the bill is forwarded to finance.
  4. Use Close Request after payment is credited.
  5. Closed requests become part of the completed billing workflow.

Information Retrieval

Reports and Search

Admin Search

Use the Admin Search screen to find employees, companies, policies, requests, and bills from one place.

Date Range Reports

Use the Reports page to enter a from date and to date, then generate employee, request, and billing reports.

Help

Messages and Troubleshooting

Situation Meaning User Action
Validation message below field The entered value is missing or does not match the rule. Correct the field and submit again.
Success popup The action completed successfully. Continue to the next workflow step.
Delete blocked The record is linked to another workflow record. Review linked policy, request, bill, or employee assignment before deletion.
Forgot password link expired The reset token is invalid, expired, or already used. Request a new reset link from the forgot password page.
